# JWT CRUD API (PHP & MySQL)

A backend-focused CRUD application built using PHP and MySQL with JWT-based authentication.

## Features
- User registration and login using JWT
- Secure API authentication via middleware
- CRUD operations for notes
- Modular backend architecture
- REST-style endpoints

## Tech Stack
- PHP
- MySQL
- JWT Authentication
- REST APIs

## Project Structure
- core/ → configuration, database, middleware
- notes/ → CRUD APIs
- login.php / register.php → authentication endpoints

## Note
This project is created for learning and demonstration purposes.
